Minhas Matérias Técnicas

Programação Web 1(Profº Jefferson Riper da Silva)

Na disciplina de Programação Web 1, aprendemos a
desenvolver sites e aplicações para a internet.
Estudamos linguagens como HTML para estruturar as páginas,
CSS para deixar o visual mais bonito e organizado,
e também conceitos iniciais de interatividade.
Essa matéria é importante porque ensina como
funcionam os sites que usamos todos os dias.

Banco de Dados (Profº Jefferson Riper da Silva)

Na disciplina de Banco de Dados 1, aprendemos a organizar, armazenar e consultar
informações de forma estruturada.
Estudamos conceitos como tabelas, registros,
chaves primárias e relacionamentos entre dados,
além de utilizar linguagens como SQL.
Essa matéria é essencial para criar sistemas
que precisam guardar informações, como cadastros e aplicativos.

Projetos de Tecnologia de Informação e Comunicação-TIC
(ProfºAfonso Henriques do Nascimento Silva)

Em Projetos de Tecnologia da Informação e Comunicação
(PTIC), aprendemos a planejar e desenvolver projetos na
área de tecnologia. Estudamos como organizar ideias,
definir objetivos, trabalhar em equipe e apresentar
soluções tecnológicas para resolver problemas.
Essa matéria ajuda a desenvolver responsabilidade,
organização e pensamento estratégico.

Sistemas embarcados e IoT (ProfºFabio Vieira Santos)

Em Sistemas Embarcados e IoT (Internet das Coisas),
estudamos dispositivos eletrônicos programáveis
que realizam funções específicas, como sensores
e microcontroladores. Também aprendemos sobre a
conexão desses dispositivos à internet,
permitindo que eles troquem informações.
Essa área é muito usada em automação residencial,
indústria e tecnologia inteligente.

Programaçãoe Algoritmos (Profª Marlene da Silva)

Na disciplina de Programação e Algoritmos,
aprendemos a criar soluções lógicas para resolver
problemas através de códigos.
Estudamos estruturas de
decisão, repetição, variáveis e lógica de programação.
Essa matéria é a base para todas
as outras áreas de desenvolvimento de software.
